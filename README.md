# Lugar Connect World

A Reddit-style community app where users can explore posts from different countries, create posts, and interact with communities.

## Features

* Browse posts by community
* Create posts with images
* Like posts
* Join communities
* User authentication

## Tech Stack

* React + TypeScript
* React Query
* Supabase
* Vite

## Future Updates

* Join/leave community functionality
* Search communities
* Full CRUD for posts (create, edit, delete)
* Improved UI and user experience

## Notes

This project was built as a full-stack learning project focusing on real-world app structure, authentication, and data handling using Supabase.
